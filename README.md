# HitTheBear
Unity3D FPS Hunting Game. 

**W,A,S,D**       Walk  
**CTRL**          Run  
**Space**         Jump  
**c key**      Crouch  
**Left Click**    Fire  
**Right Click**   Aim  
**1-6 keys**    Change Weapon  
**p key**      Silencer
